# DigitalNode Homebrew Tap

Homebrew formulas and casks for DigitalNode tools.

## Install

```bash
brew tap digitalnodecom/tap
```

### Burd

```bash
brew install burd          # the `burd` CLI
brew install --cask burd   # the Burd.app desktop app
```

### PHPless

```bash
brew install phpless
```

## Available

| Name | Type | Description |
|------|------|-------------|
| `burd` | formula | Local development environment manager (CLI) |
| `burd` | cask | Burd.app desktop application |
| `phpless` | formula | CLI for the PHPless serverless PHP platform |
